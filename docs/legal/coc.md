## Good Habits in Our Community

```
+--------------------------------------+--------------------------------------+
|  DO                                  |  DON'T                               |
+--------------------------------------+--------------------------------------+
|                                      |                                      |
|  + Ask questions respectfully        |  - Dismiss "newbie" questions        |
|                                      |                                      |
|  + Offer help without judgment       |  - Make others feel inferior         |
|                                      |                                      |
|  + Give constructive feedback        |  - Be harsh or condescending         |
|                                      |                                      |
|  + Celebrate others' wins            |  - Be jealous or dismissive          |
|                                      |                                      |
|  + Welcome diverse perspectives      |  - Discriminate or exclude           |
|                                      |                                      |
|  + Share knowledge freely            |  - Gatekeep information              |
|                                      |                                      |
|  + Be patient with learners          |  - Rush or pressure others           |
|                                      |                                      |
|  + Assume good intentions            |  - Jump to negative conclusions      |
|                                      |                                      |
+--------------------------------------+--------------------------------------+
```

## The Golden Rule

```
+-----------------------------------------------------------------+
|                                                                 |

|         Every question deserves respect                         |
|         Every contribution matters                              |
|                                                                 |
+-----------------------------------------------------------------+
```
